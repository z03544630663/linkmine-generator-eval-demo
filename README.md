# LinkMine Generator Evaluation Demo

This repository contains the static GitHub Pages demo for the BitMain, DMA and LinkMine joint natural gas generator evaluation page.

## GitHub Pages

Publish from:

- Branch: `main`
- Folder: `/ (root)`

The public demo URL will use the standard GitHub Pages format:

```text
https://<github-user>.github.io/<repository-name>/
```

## Files

- `index.html`: public demo page
- `styles.css`: page styles
- `app.js`: interactions and carousel
- `data.js`: static demo data
- `assets/`: page images and logos
